## 📌 Descripción

Este proyecto es la **OVA Final Integradora** de la asignatura Teoría General de Sistemas. Consiste en una página web que recopila y centraliza los enlaces a las 18 OVAs individuales desarrolladas por cada estudiante del curso, permitiendo acceder a todos los temas desde un solo lugar.

---

## 🎯 Objetivos

- Recopilar y organizar todas las OVAs individuales del curso en una sola plataforma.
- Facilitar el acceso a los contenidos de Teoría General de Sistemas desarrollados por los compañeros.
- Aplicar principios de diseño web (HTML, CSS, JavaScript) en la construcción de una página responsive e interactiva.

## 🛠️ Tecnologías utilizadas

- **HTML5** — Estructura de la página
- **CSS3** — Estilos, diseño responsive y animaciones
- **JavaScript (Vanilla)** — Lógica de filtros, búsqueda y renderizado dinámico
- **Google Fonts** — Tipografías Rajdhani y Space Grotesk
- **GitHub Pages** — Alojamiento web gratuito
