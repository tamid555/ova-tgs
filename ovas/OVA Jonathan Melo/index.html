Principios Fundamentales

Equifinalidad

Distintos caminos llevan al mismo estado final. En Farmatodo, el objetivo de venta se logra tanto en tienda física como en App.

Entropía

Tendencia al desgaste. Un sistema que no se innova o no recibe insumos, colapsa por desorden interno.

Sinergia

El todo es más que la suma de las partes. El éxito de Farmatodo es la unión de logística, farmacéuticos y tecnología.

Homeostasis

Capacidad de autorregulación. Ajustar el inventario automáticamente ante variaciones en la demanda.

Sist. Abiertos

Intercambian materia y energía con el entorno. Reciben proveedores y entregan productos a la sociedad.

Sist. Cerrados

Sistemas aislados con poca interacción externa; tienden rápidamente a la entropía máxima.

Retroalim.

Mecanismo de control donde la salida informa a la entrada para corregir procesos en tiempo real.

Casos Aplicados

Modelo de Sistema Abierto

Farmatodo interactúa con su entorno. Inputs: Capital, insumos, datos de mercado. Procesos: Logística y control de stock. Outputs: Productos entregados y satisfacción del cliente.

Retroalimentación y Logística

El sistema usa los datos de venta para ajustar el inventario. La información de salida (lo que se vendió) retorna como entrada para solicitar nuevos suministros, garantizando la homeostasis organizativa.
